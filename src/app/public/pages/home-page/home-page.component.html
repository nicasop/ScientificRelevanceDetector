Scientific Relevance Detector

The Document Similarity Assessment System is an advanced tool designed to analyze and visualize the similarity between academic documents from various knowledge domains. Users can manually upload a dataset in CSV format, where each record must contain critical information: the title, keywords, and abstract of the document. The platform has been specifically configured to work with documents classified in four distinct knowledge areas.

Once the user uploads the dataset, the system initiates an intelligent processing process that calculates the similarity between each pair of documents. This process is based on similarity metrics such as cosine and Jaccard. The robust processing capability enables the system to conduct a detailed assessment of the semantic relationship between the documents.

The results of this assessment are presented clearly and effectively through three visual representations: a Heat Map highlighting similarity intensities, a Two-Dimensional MDS Diagram illustrating the relative arrangement of documents in a reduced space, and a Graph revealing connections and groupings between documents. These visualizations allow users to quickly and accurately understand the similarity and relationships among the uploaded documents.

It is important to note that this system is designed to be intuitive and flexible. Users can customize their analyses, explore different similarity metrics, and adjust settings according to their specific needs. Furthermore, the system is backed by a critical approach to continuous improvement, with room for future updates and expansions based on user feedback and needs.

In summary, the Document Similarity Assessment System provides an efficient and powerful solution for evaluating similarity between documents, offering researchers and professionals a valuable tool for organizing and understanding vast sets of academic information.
